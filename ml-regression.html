Regression

Linear Regression

가장 간단한 Linear Regression 모델은 다음 식으로 표현된다.

\[\begin{equation} \hat{y} = \theta_0+\theta_1x_1+\cdots+\theta_px_p \end{equation}\]
  • \(\hat{y}\) : prediction
  • \(p\) : # of features
  • \(x_i\) : \(i_{th}\) feature
  • \(\theta_j\) : \(j_{th}\) model parameter (\(\theta_0\) for bias, \(\theta_1, \theta_2, \cdots \theta_p\) for weights)

벡터 표기를 이용하면 더욱 간단하게 표현이 가능하다.

\[\begin{equation} \hat{y} = h_{\boldsymbol{\theta}}(\mathbf{x}) = \boldsymbol{\theta}\cdot\mathbf{x} \end{equation}\]
  • \(\boldsymbol{\theta}=[\theta_0, \theta_1, \cdots, \theta_p]\) : Model parameter vector including bias & weights
  • \(\mathbf{x}=[x_0, x_1, \cdots, x_p]\) : Feature vector, \(x_0=1\)
  • \(h_{\boldsymbol{\theta}}\) : Hypothesis function with parameter \(\boldsymbol{\theta}\)

이제 \(n\)개의 훈련 세트가 있다고 해보자. 일반적으로 데이터 세트는 행이 각 데이터 레이블을 의미하고 열은 특성을 나타내므로 훈련 세트 \(\mathbf{X}\)는 \(\mathbf{X}=[\mathbf{x}^{(1)}, \mathbf{x}^{(2)}, \cdots, \mathbf{x}^{(n)}]^T\)로 나타낼 수 있다. 우리의 목표는 해당 데이터 세트가 우리의 모델에 얼마나 잘 들어맞는지 알아내는 것이다. 이러한 모델의 성능 지표로 자주 사용 되는 것으로 평균 제곱 오차 (MSE)가 있다.

\[\begin{equation} MSE\left (\mathbf{X}, h_{\boldsymbol{\theta}} \right)=\frac{1}{n}\sum_{i=1}^{n}\left(\boldsymbol{\theta}^T\mathbf{x}^{(i)}-y^{(i)} \right) ^ 2 \end{equation}\]

Normal Equation

비용 함수를 최적화하는 \(\boldsymbol{\theta}=\boldsymbol{\hat{\theta}}\)를 해석적으로 구하는 방법을 정규방정식 (Normal Equation)이라 부른다.

\[\begin{equation} \boldsymbol{\hat{\theta}}=\left(\mathbf{X}^T\mathbf{X} \right)^{-1}\mathbf{X}^T\mathbf{y} \end{equation}\]
  • \(\boldsymbol{\hat{\theta}}\) : \(\boldsymbol{\theta}\) which minimizes the cost function
  • \(\mathbf{y}=[\mathbf{y}^{(1)}, \mathbf{y}^{(2)}, \cdots, \mathbf{y}^{(n)}]\) : target vector

Proof

실제 얻는 데이터는 약간의 잡음이 섞여 있다. 따라서 모델 식에 error term을 포함하면 우리가 얻은 데이터를 표현할 수 있다.

\[\begin{equation} y_i = \theta_0 + \theta_1x_{i1} + \cdots + \theta_px_{ip} + \epsilon_i \end{equation}\]

혹은

\[\begin{equation} \mathbf{y} = \mathbf{X}\boldsymbol{\theta} + \boldsymbol{\epsilon} \end{equation}\]
  • \(\boldsymbol{\theta}=[\theta_0, \theta_1, \cdots, \theta_p]^T\) : Feature vector
  • \(\mathbf{y}=[y_1, y_2, \cdots, y_n]^T\) 
  • \(\mathbf{X}=\) \(\begin{bmatrix} 1 & x_{11} & \cdots& x_{ip}\\ 1 & x_{21} & \cdots & x_{2p}\\ \vdots & \vdots & \ddots & \vdots\\ 1 & x_{n1} & \cdots & x_{np} \end{bmatrix}\) 
\[\begin{equation} \boldsymbol{\theta^*}=\underset{\boldsymbol{\theta}}{\operatorname{argmin}} \left \|\boldsymbol{\epsilon} \right \|^2 = \left \| \mathbf{y} - \mathbf{X} \boldsymbol{\theta} \right \| ^2 \end{equation}\]

Error를 계산하면 다음과 같다.

\[\begin{align} \left \| \mathbf{y} - \mathbf{X} \boldsymbol{\theta} \right \| ^2 &= \left ( \mathbf{y} - \mathbf{X} \boldsymbol{\theta} \right ) ^T \left ( \mathbf{y}-\mathbf{X}\boldsymbol{\theta} \right ) \\ &= \left ( \mathbf{y}^T - \boldsymbol{\theta}^T \mathbf{X}^T \right ) \left ( \mathbf{y}-\mathbf{X}\boldsymbol{\theta} \right )\\ &= \mathbf{y}^T \mathbf{y} - \mathbf{y}^T \mathbf{X} \boldsymbol{\theta} - \boldsymbol{\theta}^T \mathbf{X}^T \mathbf{y} + \boldsymbol{\theta}^T \mathbf{X}^T \mathbf{X} \boldsymbol{\theta} \\ \end{align}\]

Error의 최소값을 구하기 위해 미분방정식 \(\nabla_{\boldsymbol{\theta}} \left \| \boldsymbol{\epsilon} \right \|^2 = 0\)을 푼다.

\[\begin{align} \nabla_{\boldsymbol{\theta}} \left \| \mathbf{y} - \mathbf{X} \boldsymbol{\theta} \right \| &= -2\mathbf{y}^T \mathbf{X} + 2\boldsymbol{\theta}^T \mathbf{X}^T \mathbf{X} = 0 \\ \boldsymbol{\theta^*} &= \left(\mathbf{X}^T\mathbf{X} \right)^{-1}\mathbf{X}^T\mathbf{y} \end{align}\]

이해를 위해 가장 간단한 경우인 \(p=1\), 즉 종속변수가 한개인 경우에 대해 살펴보자. 그렇다면 모델 식 및 MSE를 아래와 같이 표현할 수 있다. 편의를 위해 MSE를 손실 함수 Loss의 앞글자를 따서 L이라고 하였다.

\[\begin{align} y_i &= \theta_0 + \theta_1 x_i + \epsilon_i = \hat{y}_i + \epsilon_i \\ L &= MSE(\mathbf{x}, \hat{y}_i) = \frac{1}{n}\sum_{i=1}^{n} \left ( y_i - \theta_0 - \theta_1x_i \right )^2 \end{align}\]

Loss를 파라미터 \(\theta_0, \theta_1\)에 대해 미분하면 다음과 같다.

\[\begin{align} \frac{\partial L}{\partial \theta_0} &= -\frac{2}{n} \sum \left ( y_i - \theta_0 - \theta_1 x_i \right ) = 0 \rightarrow n\theta_0 + \left (\sum x_i \right )\theta_1 = \sum y_i \\ \frac{\partial L}{\partial \theta_1} &= -\frac{2}{n} \sum x_i \left ( y_i - \theta_0 - \theta_1 x_i \right ) = 0 \rightarrow \left (\sum x_i \right )\theta_0 + \left (\sum x_i^2 \right )\theta_1 = \sum x_iy_i \end{align}\]

행렬을 이용해 표현하면

\[\begin{equation} \begin{bmatrix} n & \sum x_i\\ \sum x_i & \sum x_i^2 \end{bmatrix} \begin{bmatrix} \theta_0\\ \theta_1 \end{bmatrix} = \begin{bmatrix} \sum y_i\\ \sum x_iy_i \end{bmatrix} \end{equation}\] \[\begin{equation} \therefore \begin{bmatrix} \theta_0\\ \theta_1 \end{bmatrix} = \begin{bmatrix} n & \sum x_i\\ \sum x_i & \sum x_i^2 \end{bmatrix} ^ {-1} \begin{bmatrix} \sum y_i\\ \sum x_iy_i \end{bmatrix} \end{equation}\]

정규방정식 테스트를 위해 \(y=4+3x\)에 error term(Gaussian Noise)을 넣어 분포를 만들었다.

import numpy as np
# Data Generation
X = 2 * np.random.randn(100, 1)
Y = 4 + 3 * X + np.random.randn(100, 1)

이제 정규방정식을 이용해 \(\boldsymbol{\hat{\theta}}\)을 구해본다.

X_b = np.c_[np.ones((100, 1)), X]   # Padding 1 to X
def normalEquation(X: np.ndarray) -> np.ndarray :
    theta_best = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(Y)
    return theta_best

theta_best = normalEquation(X_b)

>>> theta_best
array([[3.93250303],
       [3.02749724]])

우리가 원했던 값은 \(\theta_0=4, \theta_1=3\)이었으나 실제로 얻은 값은 \(\theta_0=3.9325, \theta_1=3.0275\)이다. 잡음의 영향으로 인해 약간의 오차가 발생한 것으로 보인다. 이 값을 토대로 우리의 모델을 그래프해 도시해본다.

X_new = np.array([[0], [2]])
X_new_b = np.c_[np.ones((2, 1)), X_new]
y_pred = X_new_b.dot(theta_best)

plt.plot(X_new, y_pred, 'r-')
plt.plot(X, Y, 'bo')
plt.xlabel(r'$X_{1}$')
plt.ylabel(r'Y')
plt.legend(["Prediction"])
plt.show()

# 추가할 부분

Coefficient of Determination

Feature의 개수 \(m=1\)일 때 모델의 적합성을 평가하는 지표로 주로 사용되는 것이 결정계수 (Coefficient of Determination)이다. 진행하기에 앞서 밑의 그림과 식을 살펴보자.

\[\begin{align} y_i - \bar{y} &= y_i - \hat{y}_i + \hat{y}_i - \bar{y} \\ &= (y_i - \hat{y}_i) + (\hat{y}_i - \bar{y}) \end{align}\]
  • \(y_i - \hat{y}_i\) : Difference not explained by model
  • \(\hat{y}_i - \bar{y}\) : Difference explained by model

일반적으로 error term은 평균이 0인 Gaussian Distribution을 따르므로 데이터를 평균낼 경우 error term은 사라진다. 따라서 알고 있는 데이터와 평균의 차이를 위의 두 항으로 분리할 수 있다. 이로부터 결정계수를 다음과 같이 정의하면 반응 변수 \(y_i\)의 변동량 중 모형을 통해 설명가능한 부분의 비율이라 볼 수 있다.

\[\begin{equation} r^2 = \frac{SSM}{SST} = 1 - \frac{SSM}{SSE} \end{equation}\]

여기서

  • \(SST = \sum_{i=1}^{n}(y_i-\bar{y})^2\) 
  • \(SSE = \sum_{i=1}^{n}(y_i-\hat{y}_i)^2\) 
  • \(SSM = \sum_{i=1}^{n}(\hat{y}_i-\bar{y})^2\) 
  • \(SST = SSE + SSM\) (😎 Prove!)

Prove

\(\begin{align} SST &= \sum \left (y_i - \bar{y} \right )^2 = \sum \left (y_i - \hat{y}_i + \hat{y}_i - \bar{y} \right )^2 = \sum \left ((y_i - \hat{y}_i) + (\hat{y}_i - \bar{y}) \right )^2 \\ &= \sum \left (y_i - \hat{y}_i \right )^2 + \sum \left (\hat{y}_i - \bar{y} \right )^2 + 2\sum(y_i - \hat{y}_i)(\hat{y}_i - \bar{y}) \\ &= SSE + SSM + 2 \times K \end{align}\)

\(K=0\)임을 보이자. 앞서 사용한 방식대로 error term인 SSE를 최소화하는 \(\theta_0, \theta_1\)을 구하면 된다.

\[\begin{align} \frac{\partial SSE}{\partial \theta_0} &= \frac{\partial (\sum \epsilon_i^2)}{\partial \theta_0}=2\sum\epsilon_i\frac{\partial \epsilon_i}{\partial \theta_0}=2\sum\epsilon_i(-1)=0 \rightarrow \sum\epsilon_i=0 \\ \sum \epsilon_i &= \sum y_i - n\theta_0 - \theta_1\sum x_i = 0 \rightarrow \theta_0 = \frac{1}{n}\sum y_i - \frac{\theta_1}{n}\sum x_i \\ \frac{\partial SSE}{\partial \theta_1} &= 2\sum\epsilon\frac{\partial \epsilon_i}{\partial \theta_1} = 2\sum \epsilon_i (-x_i) = 0 \rightarrow \sum x_i (y_i - \hat{y}_i) = 0 \\ x_i &= \frac{\hat{y}_i}{\theta_1} - \frac{\theta_0}{\theta_1} \rightarrow \sum \left ( \frac{\hat{y}_i}{\theta_1} - \frac{\theta_0}{\theta_1} \right ) \left ( y_i - \hat{y}_i \right ) = 0 \\ \end{align}\] \[\begin{equation} \frac{1}{\theta_1}\sum \hat{y}_i(y_i - \hat{y}_i) - \frac{\theta_0}{\theta_1}\sum(y_i - \hat{y}_i) = \frac{1}{\theta_1}\sum \hat{y}_i(y_i - \hat{y}_i) - \frac{\theta_0}{\theta_1}\sum \epsilon_i = 0 \\ \therefore \sum \hat{y}_i(y_i - \hat{y}_i) = 0 \end{equation}\]

\(K\)를 분배하면 \(\sum \hat{y}_i(y_i - \hat{y}_i) - \bar{y}\sum(y_i - \hat{y}_i)\)이므로 앞선 결과로부터 \(K=0\)임을 알 수 있다.
